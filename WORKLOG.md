# 작업 메모

## 2026-06-05 현재 상태

- MVP 파일: `outputs/ansimbodo_mvp/index.html`
- 실행 주소: `http://127.0.0.1:5180/`
- 서버 실행 예시: `python -m http.server 5180 --bind 127.0.0.1`

## 최근 변경

- `압축파일 만들기` 버튼은 `.eml` 대신 `.zip`을 생성합니다.
- 압축파일에는 `경위서.doc`와 선택 첨부자료만 들어갑니다.
- 압축 생성 후 관할 구청 담당자 메일과 `메일 주소 복사` 버튼이 표시됩니다.
- 외부 JSZip CDN 의존을 제거하고, 브라우저 내장 코드로 ZIP을 생성합니다.
- 첨부파일이 없어도 경위서만으로 압축파일을 만들 수 있게 했습니다.
- 국가배상 안내 화면은 국가배상법 제5조, 제12조와 법무부 국가배상 FAQ를 근거로 다시 작성했습니다.
- 구청 공제 접수대상이 아닌 경우, 배상 여부는 국가배상 절차에서 지구배상심의회 판단을 받아야 한다는 흐름을 명시했습니다.
- 첫 화면 정의 문구는 `시설 문제` 대신 `관리하자`로 수정했습니다.
- 국가배상도 데모에서는 메일 접수로 가정하고 `national-claim-demo@seongnam.example`을 표시합니다.
- 서명 이미지는 ZIP 첨부자료에서 제외하고, 정사각형 입력 패드로 받은 뒤 경위서 본문 우하단의 작은 서명칸에만 넣습니다.
- 경위서 저장/압축파일 생성의 필수 입력 검사를 완화했습니다. 빈 항목은 `미기재`로 출력합니다.
- 사고 지점 자동 판별을 샘플 도로명 중심에서 성남시 구·동·주요 도로명 기반 로직으로 확장했습니다.
- 도로명주소 검색 API JSONP 연동을 추가했습니다. `confmKey`가 정상 승인키이면 API 결과를 우선 사용하고, 실패 시 로컬 판별로 fallback합니다.
- `.env`의 `JUSO_API_KEY`를 로컬 서버가 읽어 `runtime-config.js`로 내려주도록 바꿨습니다. 승인키는 `.gitignore`로 커밋에서 제외합니다.
- 경위서 서명은 캔버스 전체가 아니라 실제 서명 획만 잘라내고, 표 마지막 `서명` 행의 우측 작은 칸에 들어가도록 조정했습니다.
- 보도 사고도 `하자가 뚜렷한 경우` 선택 후 도로와 같은 사고 위치 입력 화면을 거쳐 관할 구청을 찾도록 변경했습니다.
- 보도 상태 확인 화면은 2개 카드 선택 방식입니다.
  - `하자가 뚜렷한 경우` 클릭: 위치 입력 후 경위서 작성
  - `하자라고 보기 어려운 경우` 클릭: 국가배상 안내
- 사고 지점 입력 확인 버튼은 제거했습니다. 주소 입력 즉시 관할 구를 자동 반영합니다.

## 검증

- JS 문법 체크 통과.

  - `경위서.doc`
  - 선택 첨부자료
