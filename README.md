# 차량 도로사고 영조물 배상처리 MVP

영조물배상공제 원스톱서비스 아이디어를 우로보로스 Interview를 거쳐 `차량 도로사고` 중심 웹 MVP로 재구성한 산출물입니다.

## 실행

API 없이 화면만 보려면 브라우저에서 바로 열 수 있습니다.

```text
outputs/ansimbodo_mvp/index.html
```

도로명주소 API까지 확인하려면 이 폴더에 `.env`를 만들고:

```powershell
JUSO_API_KEY=승인키
```

다음 서버로 실행합니다.

```powershell
node server.mjs
```

그리고 다음 주소를 엽니다.

```text
http://localhost:5180/
```

## 현재 범위

- 첫 화면 제목: `성남시 영조물 배상공제 안내`
- 첫 화면 정의 문구에서 `시설 문제` 대신 `관리하자`를 명시
- 첫 화면을 Apple식 사진 중심 구성으로 재배치: 짧은 헤드라인, 필수요건 한 줄, 큰 선택 이미지
- 첫 화면에서 `대상 시설`, `공제 가입·등록 시설`, `관리하자 인정 필요` 반복 카드 제거
- 도로·보도 선택 영역에 두 장 좌우 배치 이미지 프레임 적용: 도로는 포트홀·싱크홀, 보도는 탈거·파손
- 도로 유형 카드에 이미지 자산 적용: 기타 도로 파손은 균열·맨홀 주변 파손, 도로 손상 없음은 눈길 미끄러짐
- 싱크홀 예시 이미지는 임시 공개 사진 자산 사용: Wikimedia Commons `Sinkhole.jpg`
- 첫 화면에서 도로 차량 사고와 보도 보행 사고 선택
- 도로 유형: 도로 꺼짐·패임, 기타 도로 파손, 도로 손상 없음
- 보도 사고 접수 전 `접수 검토 / 사진 부족 / 접수 어려움` 기준 카드
- 보도 기준 카드에 우측 이미지 적용: 하자 뚜렷함, 사진 불명확, 눈·빙판 미끄러짐
- 보도에서 하자가 안 보이는 경우 국가배상 안내로 분기
- 보도에서 하자가 뚜렷한 경우에도 도로와 같은 사고 위치 입력 화면을 거쳐 관할 구청을 자동/수동 선택
- 사고 위치를 도로명 또는 지번으로 직접 입력
- 주소에 수정구·중원구·분당구가 포함되면 관할 구 자동 표시
- 성남시 법정동·행정동 이름과 주요 도로명 기반 관할 구 자동 인식
- 여러 구에 걸치는 도로명은 구나 동이 함께 입력될 때 우선 판별
- 도로명주소 검색 API 연동: `.env`의 `JUSO_API_KEY`가 있으면 API 결과를 우선 사용하고, 실패하면 로컬 판별로 fallback
- 관할 구를 직접 선택해 도로관리팀 연결
- 관할 도로관리팀 전화번호와 시연용 이메일 표시
- 사용자가 지정한 도로관리팀 안내 번호: 수정구 `031-729-5354`, 중원구 `031-729-6354`, 분당구 `031-729-7364`
- 도로 사고 경위서 화면 내 작성
- 도로·보도 경위서에 마우스·터치 서명 캔버스 추가
- 도로 사고는 `경위서.doc`와 선택한 블랙박스 영상·현장 사진만 포함한 접수자료 압축파일 `.zip` 생성
- 압축파일 생성 후 관할 구청 담당자 메일 표시 및 복사 버튼 제공
- 블랙박스가 없을 때만 스마트도시과 CCTV 확보 요청 안내 노출
- 보도 사고 경위서 화면 내 작성
- 보도 사고는 `경위서.doc`와 선택한 현장 사진만 포함한 접수자료 압축파일 `.zip` 생성
- 작성한 도로 경위서 `.doc` 별도 저장 지원. 영상·사진 첨부 없이 저장 가능
- 경위서 작성 필수값 완화: 비어 있는 항목은 문서에 `미기재`로 표시
- 서명 입력은 정사각형 패드로 받고, 경위서 본문 우하단의 작은 서명칸에만 삽입
- 압축파일 생성은 외부 JSZip 없이 브라우저 내장 코드로 처리
- 도로 손상 없음 또는 보도 하자 안 보임 선택 시 구청 공제 접수대상이 아니며 국가배상 절차에서 별도 판단받아야 함을 안내
- 국가배상 안내는 국가배상법 제5조, 제12조와 법무부 국가배상 FAQ 기준으로 정리
- 국가배상 신청서는 국가법령정보센터 공식 배상신청서 서식으로 연결
- 성남 기준 관할: 수원지방검찰청 성남지청, 성남시 수정구 산성대로 451
- 국가배상 시연용 메일접수 주소: `national-claim-demo@seongnam.example`
- 법률판단·보상판단·CCTV 수집 금지선 표시

## 우로보로스 산출물

- `ouroboros/seed.yaml`: Gen 1 실행 Seed
- `ouroboros/evaluation.md`: Gen 1 평가와 Gen 2 입력

## 안전선

이 MVP는 실제 접수, 보험 판단, 법률 판단, CCTV 영상 수집을 하지 않습니다. 차량 피해자료는 보험사 제출 단계로 분리하고, 지자체 1차 안내에서는 경위서 작성, 관할 도로관리팀 메일 준비, 블랙박스 유무에 따른 CCTV 요청 여부만 다룹니다.
